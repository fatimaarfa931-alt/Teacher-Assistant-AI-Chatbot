# 🤖 Teacher Assistant AI Chatbot 📚

"Smart. Efficient. AI-Powered Classroom Management."

Welcome to the **Teacher Assistant AI Chatbot**, a professional-grade AI assistant designed to reduce the administrative burden on educators. This intelligent virtual assistant helps teachers automate grading, answer student queries, generate lesson plans, and manage classroom schedules seamlessly.

🚀 Live Demo
Experience the full AI assistant live on Hugging Face / Render: https://teacher-assistant-1--ArfaFatima931.replit.app

---

## ✨ Key Features

### 🧠 Intelligent Core
* **Automated Q&A:** Instantly answers frequently asked student questions about the syllabus, deadlines, and course content.
* **Lesson Planning & Quizzes:** Helps teachers brainstorm, structure daily lessons, and generate quiz questions (`test_questions.py`).
* **Grading Assistance:** Assists in evaluating objective assignments and providing quick feedback to students.

### 🎨 Clean & Modern UI
* **Interactive Dashboard:** Beautifully structured web console for easy interaction.
* **Responsive Layout:** Optimized for both desktop and mobile screens, making it easy for teachers to use on the go.
* **Smooth User Experience:** Real-time chatbot responses with clean styling and intuitive input fields.

---

## 🛠️ Tech Stack

* **Backend:** Python 🐍 & Flask 
* **Frontend:** Vanilla HTML5, CSS3, and JavaScript (ES6+)
* **Testing:** Integrated question testing via Python backend scripts.
